> Changing a remote's URL
```sh
sudo sed -i -e 's%http://.*.ubuntu.com%http://ftp.jaist.ac.jp/pub/Linux%g' /etc/apt/sources.list
```
> Updating packages, install something new
```sh
sudo apt update
sudo apt upgrade

sudo apt install -y node npm
```
